# MFA-TOTP

An easy to use, portable desktop application for TOTP (Timed One Time Password)

This app is compatible with TOTP using:
 - the Sha1 algorithm
 - 30 second time step
 - and 6 character TOTP code.
 
 Based on OTP.Net
